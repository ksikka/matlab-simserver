SimServer

Turn your MATLAB code into a web API in seconds.

View on GitHub Download .zip Download .tar.gz

SimServer is a project to enable scientists and researchers to use their existing MATLAB code from any other platform, as long as they have internet access. Use cases include developing mobile apps that talk to MATLAB, websites, web servers, you name it.

How it works

  1. Download and install SimServer on the computer where your MATLAB environment is setup.
  2. Run SimServer in your project directory. It will connect your MATLAB project to The Cloud.
  3. Go to xxx.xxx.xxx.xxx and see the automatically-generated API explorer for your MATLAB project!
